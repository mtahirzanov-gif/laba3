# Лабораторная работа №3: Система расчёта зарплат с полиморфизмом

## Описание
Программа для управления типами работ и расчёта оплаты с применением принципов ООП: наследование, полиморфизм, паттерн «Стратегия» и обработка исключений.

## Функциональность
- ✅ Добавление типов работ с базовой оплатой и процентной надбавкой
- ✅ Применение паттерна «Стратегия» для расчёта бонусов (`NoBonus`, `PercentageBonus`)
- ✅ Полиморфный расчёт итоговой оплаты через интерфейс `IWorkType`
- ✅ Валидация пользовательского ввода (строки, числа, диапазоны)
- ✅ Обработка ошибок через иерархию исключений (`PayrollException` и производные)
- ✅ Вычисление средней итоговой оплаты по всем типам работ
- ✅ Удобное консольное меню с навигацией

## Технические детали
- **Язык программирования:** C++
- **Стандарт:** C++11 и выше
- **Среда разработки:** Microsoft Visual Studio
- **Принципы ООП:**
  - **Наследование:** `IBonusStrategy` → `NoBonusStrategy`, `PercentageBonusStrategy`; `IWorkType` → `WorkTypeBase`
  - **Полиморфизм:** виртуальные методы `computePay()`, `getFinalPay()`
  - **Инкапсуляция:** приватные поля классов, публичные интерфейсы
  - **Абстракция:** чистые виртуальные функции в базовых классах
- **Паттерны проектирования:**
  - **Strategy:** выбор алгоритма расчёта бонуса во время выполнения
  - **Factory Method:** создание стратегий внутри `addWorkType()`
- **Исключения:**
  - `PayrollException` (базовый)
  - `InvalidRateException`, `DuplicateWorkTypeException`, `EmptyWorkListException`

## Как запустить в Visual Studio

### Вариант 1: Открыть существующий проект
1. Запустите **Visual Studio**
2. **Файл** → **Открыть** → **Проект или решение**
3. Выберите `.sln` / `.vcxproj` (если есть)
4. Нажмите **F5** для запуска

### Вариант 2: Создать новый проект
1. **Файл** → **Создать** → **Проект** → **Пустой проект (C++)**
2. Укажите имя: `laba3` → **Создать**
3. ПКМ по **Исходные файлы** → **Добавить** → **Существующий элемент**
4. Выберите файл `лаб3с3.cpp` из репозитория
5. **F5** — запуск

### Вариант 3: Быстрый запуск
1. Создайте пустой проект C++
2. Добавьте `лаб3с3.cpp`
3. Проверьте стандарт: **Свойства** → **C/C++** → **Язык** → **Стандарт** → `ISO C++11` или новее
4. Нажмите **Ctrl+F5** для запуска без отладки
